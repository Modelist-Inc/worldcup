Data / Setup Notes

Source: ESPN public scoreboard endpoint for FIFA World Cup: site.api.espn.com/apis/site/v2/sports/soccer/fifa.world/scoreboard.

Date range: the app requests 20260611-20260719. Change DATE_RANGE in the script if needed.

Projection: expected points = 3 × win probability + 1 × draw probability. Future match probabilities are inferred from team-strength Elo ratings, not ESPN betting odds. The model uses a logistic Elo win probability plus a soccer draw-rate adjustment. If a country is missing from the ratings table, it defaults to 1500 Elo.
